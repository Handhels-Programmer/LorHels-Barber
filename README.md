# LorHels-Barber

📱 1. El Lado del Cliente (La App Pública)
Es lo que ven tus clientes cuando entran a tu enlace. Está diseñada para ser rápida, verse como una app nativa en el celular y generar ventas.

Inicio Dinámico: Una pantalla principal atractiva donde las "Especialidades" (Cortes, Barba, etc.) se generan automáticamente a partir de lo que el administrador configure en su panel.

Galería de Estilos: Un portafolio visual para inspirar a los clientes.

Motor de Reservas Inteligente:

El cliente elige un servicio y su barbero preferido.

Calendario Anti-Choques: Al elegir una fecha, el sistema revisa la base de datos y oculta las horas que ya están ocupadas.

Panel Visual de Disponibilidad: Un cuadrito lateral que le muestra al cliente en tiempo real qué horas están libres (en gris) y cuáles ya se tomaron (tachadas en rojo).

Captura de datos (nombre y WhatsApp) para concretar la cita.

⚙️ 2. El Centro de Mando (Panel Admin)
Es el cerebro financiero y logístico del negocio. Protegido por usuario y contraseña, está dividido en 5 pestañas:

📅 Agenda y Finanzas:

Muestra cuántas citas/ventas llevas hoy y suma automáticamente el dinero a la "Caja Hoy".

Permite cambiar el estado de las citas: Pendiente -> Confirmada -> Finalizada (Cobrada) o Cancelada.

🛒 Punto de Venta (POS):

Walk-ins: Para cobrarle a clientes que llegan sin cita a cortarse el pelo.

Venta de Productos: Para vender artículos físicos (ej. ceras, aceites). Al registrar la venta, el dinero entra a la caja y el stock se descuenta automáticamente.

✂️ Gestión de Servicios: Te permite crear, editar precios o borrar los servicios de la barbería. Lo que cambies aquí, se actualiza al instante en la pantalla de inicio y en las opciones de reserva del cliente.

💈 Gestión de Staff: Puedes dar de alta a nuevos barberos o eliminarlos, actualizando el menú de los clientes al instante.

📦 Inventario (Productos): * Control de stock con botones rápidos de + y -.

Alertas automáticas: Si un producto llega a 3 unidades o menos, te lanza una alerta visual roja de "Stock Bajo".
